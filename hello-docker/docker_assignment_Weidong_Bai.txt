Question 2.1 — Why pin?

If you don't pin a version, pip install pandas grabs whatever's latest at build time. Six months later someone rebuilds your image and gets a newer pandas that behaves differently — same Dockerfile, different results, no error message to explain why.

Question 2.2 — Recipe or cake?

The Dockerfile is better because it's readable. Anyone can open it and see exactly what was installed and why. A pre-built image is a black box — you just have to trust it.
